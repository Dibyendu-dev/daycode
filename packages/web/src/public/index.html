Everything you need,
right in the terminal

Daycode combines the power of modern AI with the simplicity of the command line. No IDE required, no context switching — just you and your terminal.

Multi-Model AI

Switch between Gemini, Claude, GPT, and DeepSeek on the fly. Each model brings different strengths to your workflow.

Real-Time Streaming

See AI responses stream in character by character. Low-latency, real-time chat experience built on SSE and Hono.

Terminal Native

Built with OpenTUI, not Electron. Runs entirely in your terminal with zero overhead. No IDE, no browser tab needed.

OAuth Authentication

Secure login with Clerk OAuth. Your sessions and preferences sync across devices with zero-config setup.

Credit-Based Billing

Pay only for what you use. Integrated with Polar.sh for usage metering and Stripe checkout. Transparent pricing.

Smart Modes

PLAN mode for read-only code analysis, BUILD mode for file manipulation and command execution. Always in control.

Download the binary and start a session in seconds. No signup walls, no complex setup.
